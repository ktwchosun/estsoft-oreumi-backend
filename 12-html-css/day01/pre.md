### Heading 3

HTML 문서는 항상 아래 구조로 작성해야 한다.

`(backtick)` 기호를 세 개 이상 써서 코드 블록을 작성할 수 있다.

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>애국가</title>
</head>
<body>

    <h1>애국가</h1>
    <p>
        동해물과 백두산이 마르고 닳도록
        하느님이 보우하사 우리나라 만세
    </p>
    <p>
        무궁화 삼천리 화려 강산<br>
        대한 사람 대한으로 길이 보전하세
    </p>

    <hr>
    
</body>
</html>
```
